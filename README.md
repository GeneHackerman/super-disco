# super-disco

* This application is a work scheduler with date and time implemented via moment.js. Any tasks set within the 9-5 business day will be time logged from past(grey), present(red), and future(green) through color coded settings. These colors are synched to the asynchronous clock function. Each hour can be edited and logged via localStorage courtesy of the save button(save icon with blue background) to the right of the text area.
